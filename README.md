# Comanda na Mão V6.3 Visual

Essa versão melhora a V6.2 visual com:
- menu lateral recolhível
- visual mais SaaS / produto
- animações suaves com framer-motion
- dashboard mais forte
- cards e seções mais elegantes
- visual mais consistente

## Uso
Substitua a pasta `frontend` da sua V6 atual por esta nova `frontend`.

## Passos
```bash
cd frontend
npm install
npm run dev
```
